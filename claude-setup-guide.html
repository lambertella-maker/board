First — what even is the Terminal?

Your Mac has a secret text-based way to give it instructions — called the Terminal. Instead of clicking buttons, you type commands. That's all it is. Think of it like texting your computer instead of tapping on icons.

It sounds scary but it's not. Every command you'll type in this guide is shown exactly as you need to type it. You literally just copy and paste.

💡
The golden rule of Terminal If something goes wrong, nothing explodes. You can always close the window and start fresh. You cannot break your Mac by typing the wrong thing in this guide.

Open Terminal

The easiest way is to use Spotlight Search — the little magnifying glass built into every Mac. Press ⌘ Cmd + Space at the same time, then type "terminal" and press Enter.

You'll see a black (or dark) window appear. This is your Terminal. There will be a line of text with a $ at the end — that's just the Terminal saying "I'm ready, what do you want me to do?"

Your Terminal might look different The colours, font, and exact text will vary. That's fine. The important bit is the % or $ at the end of the line — that means it's ready for you.
2
Check if Node.js is installed

Claude Code runs on something called Node.js — think of it as the engine that makes it go. Your Mac might already have it. Let's check.

Click inside the Terminal window, then type this exactly and press ↵ Return:

$ node --version
← type this

If you see a version number (like v20.11.0) — great, Node is installed! Skip straight to Step 3.

Terminal
yourname@MacBook ~ % node --version v20.11.0   yourname@MacBook ~ %

If you see an error (like "command not found") — you need to install Node.js. It's free and easy, no Terminal needed for this bit:

N
nodejs.org/en/download
Download Node.js
Free, open-source JavaScript runtime. Click the macOS installer button and run the .pkg file that downloads — it's just like installing any other app.
⬇️ Download for macOS →
  1. Go to nodejs.org and click the big green download button
  2. Open the downloaded file (it'll be in your Downloads folder — looks like a box icon)
  3. Click through the installer — just keep pressing "Continue" and "Install"
  4. Once it's done, go back to Terminal and try node --version again
⚠️
Still says "command not found" after installing? Close your Terminal window completely, re-open it (Cmd+Space → terminal), and try again. The installer sometimes needs a fresh window to take effect.
3
Install Claude Code

Now for the main event. This one command downloads and installs Claude Code on your Mac. Copy and paste it into your Terminal, then press ↵ Return:

$ npm install -g @anthropic-ai/claude-code
← copy this exactly

You'll see a lot of text scroll past — that's normal, it's just telling you what it's doing. Wait until it stops and you see the % prompt again.

Terminal
yourname@MacBook ~ % npm install -g @anthropic-ai/claude-code   added 47 packages in 4s   13 packages are looking for funding run `npm fund` for details   yourname@MacBook ~ %
🔐
If it asks for your password It might say something like sudo npm install... if the first one fails. Type your Mac login password — it won't show dots or anything as you type, that's normal — then press Return. This is your Mac asking for permission to install software, the same as any other installer.
✅
How to know it worked When you see the % prompt return and no big red error messages, it worked. A few warning messages in amber/yellow are fine — those aren't errors.
4
Tell Claude where to work

Claude Code works inside folders on your Mac — like a project folder. Before you start it, you navigate Terminal to the folder you want Claude to help you with.

The command for this is cd which stands for "change directory" (directory is just tech-speak for folder).

The easiest way: open Finder, find the folder you want to use, then drag that folder directly into the Terminal window. It'll automatically fill in the path for you. Then press Return.

Terminal
# Example: you dragged your "My Project" folder in yourname@MacBook ~ % cd /Users/yourname/Desktop/My\ Project   # Notice the folder name appears in the prompt now: yourname@MacBook My Project %
📁
Just starting out? Use your Desktop. If you don't have a specific project folder yet, just navigate to your Desktop: type cd ~/Desktop and press Return. Claude will be able to create and read files there.
5
Start Claude

This is the moment. Type this and press ↵ Return:

$ claude
← just "claude"

The first time you run it, Claude will ask you to log in. It'll show you some options — usually it'll open a browser window for you to sign in with your Anthropic account. Follow whatever it says on screen.

Claude Code
yourname@MacBook Desktop % claude   ╔═══════════════════════════════╗ ║ Claude Code v1.x.x ║ ╚═══════════════════════════════╝   ✦ Welcome! Let's get you set up. Opening browser for authentication...   Waiting for login

Once you're logged in, the browser will close and Terminal will come alive with Claude's interface. You'll see a message prompt — just like a chat window, but in your Terminal.

Claude Code — Desktop
✓ Logged in successfully!   Working directory: /Users/yourname/Desktop   ┌────────────────────────────────────────────────────────────────┐ │ What would you like to do? │ └────────────────────────────────────────────────────────────────┘   >
🎉
You're in! That > is Claude waiting for you. Just type your question or task in plain English — exactly how you'd ask a person. "Write me a short poem about my cat", "Can you help me write an email to my landlord", whatever you need.
6
What to actually say to Claude

Claude Code is a lot like Claude on the web — just talk to it normally. You don't need special commands. Here are some examples of things you can type straight in:

Claude Code — examples
# Asking for help with writing: > Can you proofread this email draft for me?   # Asking about code: > I have a Python file on my Desktop called app.py, can you explain what it does?   # Just chatting: > What's the best way to learn JavaScript as a complete beginner?   # Creating files: > Create a simple to-do list as an HTML file on my Desktop

When Claude is done with its response, it'll show the > prompt again so you can ask something else. It's a conversation — you can follow up, ask it to change things, or start a completely new topic.

7
The 4 things you actually need to know
🚪
How to exit Claude

Type /exit and press Return, or press Ctrl+C twice. This takes you back to the regular Terminal prompt.

🛑
Stop Claude mid-response

If Claude is typing a really long answer and you want it to stop, press Ctrl+C once.

⬆️
Re-run a previous command

In the regular Terminal (not inside Claude), press the ↑ Up arrow to scroll through commands you've used before.

🔄
Start a fresh conversation

Each time you type claude in a new Terminal session, you start a brand new conversation. Claude won't remember the previous chat.

📂
Change which folder you're in

Exit Claude first (/exit), then type cd ~/Desktop or drag a folder into Terminal, then type claude again.

✏️
Multi-line messages

To write a longer message across multiple lines, press Shift+↵ Return to go to a new line without sending.

!
If something goes wrong
❌
"command not found: claude" The installation didn't complete fully, or the Terminal needs to be refreshed. Close the window, open a new Terminal, and try the npm install -g @anthropic-ai/claude-code command again.
❌
"EACCES: permission denied" Try adding sudo in front: sudo npm install -g @anthropic-ai/claude-code. It'll ask for your Mac password.
❌
"node: command not found" after installing Node.js Close Terminal completely, reopen it, and try again. If it's still not found, try restarting your Mac — Node updates your PATH settings and sometimes needs a restart.
💬
Still stuck? Copy the exact error message from your Terminal and paste it into Claude at claude.ai (the normal web version). Ask "what does this error mean and how do I fix it?" — Claude is very good at diagnosing its own installation problems.
